# pytorch-pensieve
Implementing [pensieve](http://web.mit.edu/pensieve/) using pytorch.

The [author](https://github.com/hongzimao/pensieve) of pensieve uses tensorflow to implement it.


run an example:

        ```
        python main.py
        ```

## dependencise:

- python-2.7.15
- pytorch-1.0.1
- numpy-1.16.2
